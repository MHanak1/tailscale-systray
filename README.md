# tailscale-systray

Linux port of tailscale system tray menu.

![tailscale-systray](/screenshot.png)

## Usage

```
$ tailscale-systray
```

## Requirements

* tailscale

## Installation

Building app require gcc, libgtk-3-dev, libappindicator-3-dev

```
sudo apt-get install gcc libgtk-3-dev libappindicator3-dev
```
